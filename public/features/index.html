Features

Everything you need to create and share presentations, nothing you don't.

Markdown-based

Write in markdown — the fastest way to create content. No drag-and-drop, no layout tools.

Browser-based

Runs entirely in your browser. No installation, no plugins, no Flash.

Shareable links

Save your presentation and share a link. Viewers don't need an account.

GitHub Gist integration

Save presentations to GitHub Gists for free hosting and version control.

Mobile responsive

View your slideshows on any device — desktop, tablet, or phone.

Keyboard navigation

Use arrow keys or swipe to navigate slides. Press Escape for overview.

Code syntax highlighting

Beautiful code blocks with syntax highlighting for developer presentations.

AI-compatible

Generate markdown with ChatGPT, Claude, or Gemini, then paste it in.

Free forever

No pricing tiers, no paywalls, no signup required.
