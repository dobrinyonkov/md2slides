About md2slides

Making presentations simple, one markdown file at a time.

Why md2slides?

Presentations should be about your content, not fighting with software. We built md2slides because we were tired of:

  • Spending hours adjusting slide layouts
  • Compatibility issues between PowerPoint versions
  • Needing an account just to view someone's presentation
  • Clunky export processes

How it works

Write your content in markdown — the same text format developers use for documentation. Use --- to separate slides. Paste it into md2slides and your presentation is ready to share.

Built for everyone

Whether you're a developer who lives in markdown, a teacher creating lesson materials, or a startup founders pitching investors — md2slides works for you. No learning curve, no account needed, no cost.

Open source

md2slides is built with transparency in mind. The entire stack runs in your browser — your markdown never touches a server unless you choose to save and share it via gists.
